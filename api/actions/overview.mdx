## Overview

Actions represent operations that can be performed on the NEAR blockchain. NEAR Connect defines a set of `ConnectorAction` types that provide a standardized interface for various blockchain operations.

## ConnectorAction Type

All action types are unified under the `ConnectorAction` union type:

```typescript
type ConnectorAction =
  | CreateAccountAction
  | DeployContractAction
  | FunctionCallAction
  | TransferAction
  | StakeAction
  | AddKeyAction
  | DeleteKeyAction
  | DeleteAccountAction
  | UseGlobalContractAction
  | DeployGlobalContractAction;
```

## nearActionsToConnectorActions

Converts NEAR JS SDK actions to NEAR Connect connector actions.

### Signature

```typescript
function nearActionsToConnectorActions(
  actions: (Action | ConnectorAction)[]
): ConnectorAction[];
```

### Parameters

<ParamField path="actions" type="(Action | ConnectorAction)[]" required>
  Array of NEAR JS SDK `Action` objects or `ConnectorAction` objects to convert
</ParamField>

### Returns

Returns an array of `ConnectorAction` objects with normalized formats.

### Example

```typescript
import { nearActionsToConnectorActions } from "@near-connect/core";
import { functionCall, transfer } from "@near-js/transactions";

const nearActions = [
  functionCall({
    methodName: "set_greeting",
    args: { message: "Hello" },
    gas: BigInt(30000000000000),
    deposit: BigInt(0),
  }),
  transfer(BigInt(1000000000000000000000000)), // 1 NEAR
];

const connectorActions = nearActionsToConnectorActions(nearActions);
// [
//   {
//     type: "FunctionCall",
//     params: {
//       methodName: "set_greeting",
//       args: { message: "Hello" },
//       gas: "30000000000000",
//       deposit: "0"
//     }
//   },
//   {
//     type: "Transfer",
//     params: { deposit: "1000000000000000000000000" }
//   }
// ]
```

### Behavior

- If an action already has a `type` property (is already a `ConnectorAction`), it is returned as-is
- NEAR JS SDK actions are converted to their corresponding `ConnectorAction` format
- Function call arguments are automatically deserialized from `Uint8Array` to JSON objects
- BigInt values (gas, deposit, stake) are converted to string format
- Public keys are converted to their string representation
- Throws an error if an unsupported action type is encountered
